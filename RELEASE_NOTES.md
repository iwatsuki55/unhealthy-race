# リリースノート

## アンヘルシーレース（仮） 現行版

アンヘルシーレース（仮）の現行版を本番反映しました。
不健康行動と健康行動を記録しながら、ポイントの増減とレース形式の比較を通じて生活習慣の乱れを見える化できる状態です。

## 主な機能

- メール登録 / ログイン
- 初回プロフィール設定
- 不健康行動 / 健康行動の登録
- ポイント計算と制限ルール
- ホーム、履歴、今週の推移表示
- レース開始による実質リセット
- 招待コードによる友達 / ライバル参加
- レース参加メンバーのランキング表示
- `🥇🥈🥉` つきの順位演出
- コメントトーン切り替え
- 共有用サマリー
- 終了予定日、終了待ち、終了トリガーの見える化
- 前回レース結果表示

## レースの仕様

- レースは個人でも開始可能です。
- レース期間は 7日 / 14日 / 30日 から選択できます。
- 新しいレースを開始すると前回レースは終了扱いになります。
- 過去ログは削除されず残ります。
- 終了予定日を過ぎると `終了待ち` 表示になります。
- 終了待ち中でも記録は継続できますが、順位とポイントは暫定です。
- オーナーが次のレースを開始すると結果が確定します。
